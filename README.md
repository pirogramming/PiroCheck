# PIROCHECK 🔍


PiroCheck는 동아리 출결 및 과제 관리를 효율적으로 돕는 통합형 플랫폼입니다.  
운영진의 번거로운 수작업을 줄이고, 부원들의 출결과 과제를 한눈에 확인할 수 있도록 지원합니다.

---

## 🚀 주요 기능

- **✅ 실시간 출결 코드 생성 및 기록**
- **📚 과제 관리 및 상태 추적**
- **🖥 관리자 / 사용자 모드 분리**
- **📈 직관적인 UI/UX로 학습 효율성 증가**
